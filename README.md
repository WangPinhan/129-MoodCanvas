# MoodCanvas · 心象日记

> **把文字情绪，转化为可视化的心象风景**  
> 一款面向年轻用户的轻量级情绪记录与疗愈应用，基于 Qt + 大模型语义分析 + 动态图形渲染，帮助你温柔地看见自己的情绪变化。

---

## ✨ 项目简介

**MoodCanvas（心象日记）** 不是普通的日记本，而是一个「情绪可视化」工具。  
每天只需一句话、一个表情或几个标签，系统就会将你的文字转化为多维情绪向量，并生成一张独一无二的 **动态心象画布**——颜色、流动速度、粒子效果、星球形态都会随你的情绪实时变化。

项目强调 **私密性**（默认本地存储）、**低门槛记录** 和 **有温度的反馈**，旨在让用户在记录与回看中，自然地觉察情绪模式，而非依赖复杂的社交或诊断功能。

---

## 🎯 核心功能

| 功能模块 | 说明 |
|---------|------|
| **今日心象输入** | 支持文字、表情、情境标签三种记录方式，一键生成情绪分析结果（情绪类别、愉悦度/激活度、关键词、触发因素、安慰语）。 |
| **动态心象画布** | 基于情绪向量实时渲染动态背景（颜色渐变、粒子流动、波动频率），可截图保存，是项目的核心创意实现。 |
| **时间长廊** | 月历视图，每天以色块/星球展示情绪主色；点击可查看当日所有日记原文、分析详情和心象截图。 |
| **每周情绪报告** | 自动统计本周情绪分布（环形图）、愉悦度/激活度趋势曲线、高频关键词、触发因素标签，并生成自然语言小结与下周建议，支持导出 PNG/PDF。 |
| **解忧杂货铺** | 轻量情绪辅助工具：ABC 小卡片、想法重构建议、呼吸练习引导、安慰语收藏等（非医疗诊断，仅提供陪伴感）。 |
| **匿名树洞** | 可选匿名分享心象画面 + 模糊心情，由大模型生成回响（支持敏感词过滤，后期可扩展）。 |
| **心情小怪兽（桌宠风格）** | 将情绪拟人化为小怪兽（焦虑怪、拖延怪、开心怪等），提示用户「理解而非消灭」情绪。 |
| **今日植物（情绪花园）** | 根据当天最后一条心情记录，植物呈现不同生长状态（开花、卷叶、下雨等），温柔回应情绪。 |
| **隐私与数据管理** | 本地 SQLite/JSON 存储，支持数据导出、API Key 配置、模型连接测试，清晰说明数据流向。 |

## 🛠 技术栈

- **前端框架**：Qt (C++ / QML)  
- **动态图形渲染**：OpenGL / Canvas / WebGL（自定义粒子系统与渐变映射）  
- **语义分析**：大模型 API（OpenAI 兼容接口）或 Hugging Face 情绪分类模型，自定义中文 prompt 与结构化 JSON 解析  
- **数据存储**：SQLite / LocalStorage（本地日记、情绪向量、聚合统计）  
- **可视化图表**：Qt Charts / 第三方图表库（环形图、折线图）  
- **导出功能**：PNG / PDF 生成

## 🚀 快速开始

### 环境要求
- Qt 5.15+ / Qt 6.x
- C++17 编译器
- 网络连接（用于大模型 API，可选）

### 配置
1. 克隆仓库
2. 打开项目文件（`.pro` 或 `CMakeLists.txt`）
3. 在「设置/隐私」界面配置 API Key、接口地址和模型名称，并测试连接
4. 运行应用

---

## 📦 数据存储与隐私

- 所有日记、心象记录默认保存在 **本地**（SQLite / JSON 文件）
- 仅在调用大模型分析时，发送必要的文本内容（不含个人身份信息）
- 支持一键导出全部数据，便于备份或迁移
